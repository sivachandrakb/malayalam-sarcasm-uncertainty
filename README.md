# Uncertainty-Aware Malayalam Sarcasm Detection

An uncertainty-aware multilingual NLP system for Malayalam sarcasm classification using Transformer architectures, Monte Carlo Dropout, and Evidential Learning.

## Live Demo

Hugging Face Spaces:
https://huggingface.co/spaces/sivachandrakb/sarcasm_detection

## GitHub Repository

https://github.com/sivachandrakb/malayalam-sarcasm-uncertainty

## Features

- Multilingual sarcasm detection
- Transformer-based NLP inference
- Monte Carlo Dropout uncertainty estimation
- Predictive entropy visualization
- Real-time Gradio deployment
- Hugging Face Spaces hosting

## Model Architecture

Input Text
↓
Transformer Encoder
↓
Evidential Classification Head
↓
MC Dropout Sampling
↓
Prediction + Uncertainty Estimation
